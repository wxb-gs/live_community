选择前端 UI 技术栈

小红书风格 = 瀑布流布局 + 大图封面卡片 + 流畅交互

A

Vite + React + Tailwind CSS

轻量级，完全自定义设计，Tailwind 原子化样式灵活度高。瀑布流可用 react-masonry 或手写 CSS columns 实现。适合追求像素级还原小红书设计。

B

Vite + React + Ant Design

阿里系组件库，和 Spring Cloud Alibaba 技术栈呼应。Card/Upload/Modal 等组件开箱即用，但风格偏企业级，定制小红书风格需要大量覆盖样式。

C

Next.js + Tailwind CSS

支持 SSR/SSG，SEO 友好。但后端已是独立部署的微服务，Next.js 的 Node Server 会多一层。适合未来需要 SEO 的社区内容页，但当前阶段稍重。

D

Vite + React + CSS Modules

纯粹的轻量方案，无第三方 UI 库依赖。样式完全自主控制，适合打造自有设计体系。但轮子需要自己造（瀑布流、骨架屏、弹窗等）。
