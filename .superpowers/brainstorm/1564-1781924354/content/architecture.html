整体架构设计

三层结构：前端 SPA → Gateway REST API → 后端微服务

页面路由结构（React Router v6）
/                    → FeedPage (瀑布流首页)
/note/:noteId        → NoteDetailPage (笔记详情 + 评论)
/publish             → PublishPage (发布笔记)
底部 TabBar 导航：
🏠 发现 | ➕ 发布 | 💬 消息 (预留)
组件树
App
├── TabBar (底部导航)
├── FeedPage
│   ├── WaterfallLayout
│   │   └── NoteCard[] (卡片)
│   └── LoadingSkeleton
├── NoteDetailPage
│   ├── ImageHero (大图)
│   ├── NoteContent
│   ├── CommentList
│   │   └── CommentItem[]
│   └── CommentInput
└── PublishPage
    ├── ImageUploader
    ├── TitleInput
    └── ContentEditor
关键技术决策
构建工具Vite 5
UI 框架React 18 + Tailwind CSS 3
路由React Router v6
状态管理React 内置 (useState + useContext) — 3页面无需引入 Redux
HTTP 客户端fetch API (封装 api.ts) — 零额外依赖
瀑布流CSS columns 方案 — 纯 CSS 实现，无额外库
